# width — Get width of image

## 描述

> public Intervention\Image\Image width()

Returns the width in pixels of the current image.

## 参数

none

## 返回值
Width of current image as integer

## 示例

```php
// read width of image
$width = Image::make('public/foo.jpg')->width();
```
